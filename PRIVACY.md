# Privacy Policy

Fuel Cost for Maps does not create accounts, track browsing history, or sell data.

## Data Stored Locally

The extension stores settings in browser extension storage:

- selected country, fuel type, currency, and fuel economy;
- optional saved Dutch license plate, only when "Save plate locally" is enabled.

Saved settings remain on the user's device/browser profile unless browser sync is enabled by the browser.

## Data Sent to the Backend

The extension requests fuel prices from the Fuel Cost API using selected country and fuel type. For optional Dutch vehicle lookup, the selected country and entered license plate are sent to the backend so it can query RDW open data.

## Third-Party Data Sources

- European Commission Weekly Oil Bulletin for fuel prices.
- RDW open data for optional Netherlands vehicle economy lookup.

## Analytics and Advertising

The extension includes no analytics, advertising SDKs, or remote user tracking.
